# Python Data Automation Script

A simple Python script for automating CSV data cleaning tasks.

## Features
- Reads CSV input files
- Removes leading and trailing whitespace from fields
- Outputs a clean CSV file automatically

## How to run
1. Place your CSV file as `input.csv` in the same directory
2. Run the script:
   ```bash
   python script.py
3. The cleaned file will be saved as output_clean.csv

## Use case
Useful for cleaning CSV data before importing into databases, spreadsheets, or other systems where consistent formatting is required.
